Installation: Famous 2-minute install

  1. Unzip the package in an empty directory and upload everything.
  2. Open install in your browser. It will take you through the process to set up a .env file with your database connection details.
    1. If for some reason this doesn’t work, don’t worry. It doesn’t work on all web hosts. Open up .env.example with a text editor like WordPad or similar and fill in your database connection details.
    2. Save the file as .env and upload it.
    3. Open install in your browser.
  3. Once the configuration file is set up, the installer will set up the tables needed for your site. If there is an error, double check your .env file, and try again. If it fails again, please go to the support forums with as much data as you can gather.

System Requirements (Free and Professional)

  • PHP version 5.6.
  • MySQL.
  • The OpenSSL PHP module.
  • The PDO PHP module.
  • The mbstring PHP module.
  • The Tokenizer PHP module.
  • The mod_rewrite, mod_gzip, mod_filter and mod_expires modules.
  • The ionCube Loader PHP module. Use Loader Wizard for easy installation

System Requirements (Enterprise)

  • PHP version 5.2.4 or higher.
  • MySQL.
  • The OpenSSL PHP module.
  • The PDO PHP module.
  • The mbstring PHP module.
  • The Tokenizer PHP module.
  • The mod_rewrite, mod_gzip, mod_filter and mod_expires modules.

Social Media API Settings

You can enter your Social media credentials in Admin panel in API Credentials page.
  • Facebook - for Facebook login.
  • Google+ - for Google+ login. Please enable, "Google+ API" in Google Developer Console. Redirect URIs: URL/googleAuthenticate
  • Google API Key - for Google Map. Please enable, "Google Maps JavaScript API" and "Google Static Maps API" in Google Developer Console.
  • LinkedIn - for LinkedIn connect.

Email(SMTP) Settings

You can enter your SMTP credentials in Admin panel in Email Settings page.

Cron Jobs

You have to configure the below Cron Job in your server.
  • * * * * * php PATH/artisan schedule:run >> /dev/null 2>&1
  • The above Cron Job is for updating Currency Rates(Daily), iCal Synchronization(Hourly), updating Expired Reservations(Hourly), Updating Travel Credit(Daily) and Task Queueing(Every Minute).
  • And the above Cron Job is for Files, Database and Cleanup backup(Monthly, Daily and Monthly). Backup files will be saved in PATH/storage/app/ folder.
